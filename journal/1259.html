Translating Epicurus

by Peter Saint-Andre

2008-08-17

I've decided to start a translation of the extant writings of Epicurus, inspired most recently by some thoughts on ataraxia posted by Manual Nunez over at Sunni Maravillosa's website. But don't expect me to finish anytime soon, for several reasons:

  • I'm ridiculously busy in my real job, and will be for the foreseeable future.
  • I want to publish both an English translation and the original Greek, but I haven't been able to find the Greek text online anywhere, so I'm painstakingly typing it in character by character (with full diacritics).

But finish I shall, eventually. And naturally I will make the resulting text available in the public domain.

(So far all I've done is work on preliminary versions of the first few paragraphs of the Principal Doctrines -- you can find the work-in-progress here.)
