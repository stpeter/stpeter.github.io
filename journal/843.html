JMTP?

2006-01-12

After my talk at the North Colorado Linux Users Group the other night, I got to chatting with David Van Maren about the possibility of using the Jabber/XMPP server-to-server protocol (which includes reverse DNS lookups and, beyond that, strong server authentication if you support TLS) for exchanging inter-domain email. It's an intriguing idea, which I immediately dubbed JMTP (the Jabber Mail Transport Protocol). We've talked about this in the past but never really worked on it. One of these days I'll give it some more serious thought. ;-)

Update: Naturally I'm not the first one to think of this: both Sander Devrieze and Robert Mibus have written or coded some thoughts, and I'm sure they're not alone.
