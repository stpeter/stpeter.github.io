Something Beautiful

by Peter Saint-Andre

2008-05-08

I'm working all the time these days, so I don't have the time to write blog entries of a contemplative nature. But I wanted to acknowledge a beautiful post I found recently at Arthur Silber's blog. I don't know whether the post is primarily about music, politics, ethics, or philosophy -- all four (and more), I think, which to my mind puts it in the same class as the essays of Glenn Gould (and that, for me, is high praise). So thank you, Arthur, for writing so well, caring so deeply, and thinking so independently.
